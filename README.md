# ✨ LUXE HAIR AI

AI 기반 프리미엄 헤어스타일 가상 체험 서비스

## 🌟 Features

- 📸 사진 업로드로 새로운 헤어스타일 미리보기
- ✂️ 컷, 펌, 염색 다양한 카테고리
- 🎨 커스텀 스타일 사진 업로드 지원
- 🔄 Before/After 비교 슬라이더
- 💫 골드 테마의 럭셔리 UI/UX

## 🚀 Quick Start

1. [Replicate](https://replicate.com)에서 API 키 발급
2. 웹사이트에서 내 사진 업로드
3. 원하는 스타일 선택
4. API 키 입력 후 변환!

## 🔧 Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Replicate AI API
- GitHub Pages
